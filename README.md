# blog

> A Vue.js project

## 运行

``` bash
注: 本地必须依据安装了mongodb并且启动了服务

# install dependencies
npm install

# start up server
npm run dev-server

# start up client <!-- localhost:8081 -->
npm run dev-client

# start up admin <!-- localhost:8083,  -->
# account: admin password: admin
npm run dev-admin 
 
```

## 注

	－ 目前的版本适合express和vue的初学者学习
