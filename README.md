# blog

> A Vue.js project

## 版本

vue: 2.1.8
vuex: 2.1.1
vue-router: 2.1.1

## 运行

``` bash
注: 本地必须依据安装了mongodb并且启动了服务

# install dependencies
npm install

# start up server
npm run dev-server

# start up client <!-- localhost:8081 -->
npm run dev-client

# start up admin <!-- localhost:8083,  -->
# account: admin password: admin
npm run dev-admin 
 
```
